# SCT_DataTrack_Task03
 
📊 Bank Marketing Decision Tree Classifier

This project builds a Decision Tree Classifier using the [Bank Marketing Dataset](https://archive.ics.uci.edu/ml/datasets/Bank+Marketing) to predict whether a customer will subscribe to a term deposit based on demographic and behavioral data.

---

## 📌 Objective

Use a decision tree to:
- Analyze customer attributes
- Predict marketing campaign success (term deposit subscription)

---

## 📂 Dataset

**Source:** UCI Machine Learning Repository  
**Filename:** `bank-additional-full.csv`  
- Contains 41,188 records and 20 input attributes  
- Target variable: `y` (Yes/No for subscription)

---

## 🧠 Model Used

- `DecisionTreeClassifier` from `scikit-learn`
- Train-test split: 80/20
- Evaluation: Accuracy, Classification Report

---

## 🛠️ Tools & Libraries

- Python
- Pandas
- Scikit-learn
